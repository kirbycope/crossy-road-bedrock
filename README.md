# crossy-road-bedrock
Mysticat's Crossy Road for Minecraft Bedrock Edition.

## Installation
1. Download the [mctemplate](https://github.com/kirbycope/crossy-road-bedrock/raw/main/crossy-road-bedrock.mctemplate)
1. Double-click the mctemplate file
1. Load world
